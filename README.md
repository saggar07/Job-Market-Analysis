# Job-Market-Analysis

## Project Overview
This project analyzes the global data jobs market using the data_jobs dataset by Luke Barousse.
The objective is to understand job demand, salary trends, required skills, job roles, and work formats (remote vs on-site) in the data industry.
The analysis helps aspiring data professionals and stakeholders answer key questions such as:
Which data roles are most in demand?
What skills are most frequently required?
How salaries vary across roles, locations, and experience levels
The impact of remote work on compensation
This project follows a real-world data analytics workflow, including data cleaning, exploratory data analysis (EDA), and insight-driven visualizations.
